# Client

Frontend for the AI PDF Assistant.

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui
- Axios

---

## Install

```bash
npm install
```

---

## Run

```bash
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

## Features

- Upload PDFs
- Chat interface
- Responsive UI
- Dark mode support
- Sidebar navigation
- AI responses
- Upload new PDFs during chat
